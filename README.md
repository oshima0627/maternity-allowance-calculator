# 出産手当金シミュレーター

妊娠中の女性が出産手当金の支給額を簡単にシミュレーションできるWebアプリケーションです。

## 🚀 機能

### 💰 正確な計算
- 厚生労働省の公式データに基づく計算ロジック
- 標準報酬月額等級表による正確な計算
- 単胎・多胎妊娠の期間差対応

### 📊 詳細な結果表示
- **支給期間**: 産前産後期間の詳細表示
- **出産手当金**: 標準報酬日額・出産手当金日額・総支給額
- **現在との比較**: 通常時の手取りとの比較表示
- **手取り維持率**: 現在の手取りに対する手当金の割合

### 🎯 ユーザー体験
- **シンプル入力**: 月額給与・出産予定日・妊娠タイプの3項目のみ
- **リアルタイム計算**: 入力と同時に結果を表示
- **レスポンシブデザイン**: PC・スマホ完全対応
- **アクセシビリティ**: キーボード操作・スクリーンリーダー対応
- **FAQ**: よくある質問10項目

## 🛠️ 技術仕様

### フロントエンド
- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイル**: CSS Modules + CSS Custom Properties
- **状態管理**: React Hooks

### 計算ロジック
- **標準報酬月額**: 健康保険・厚生年金保険の保険料額表
- **給付金**: 標準報酬日額の2/3
- **支給期間**: 単胎42日+産後56日、多胎98日+産後56日

## 📋 計算基準

### 使用データ（2025年度）
- **標準報酬月額等級表**: 厚生労働省公式データ
- **給付率**: 標準報酬日額の2/3
- **支給期間**: 産前42日（多胎98日）+ 産後56日

### 計算内容
```
標準報酬日額 = 標準報酬月額 ÷ 30
出産手当金日額 = 標準報酬日額 × 2/3
総支給額 = 出産手当金日額 × 支給日数
```

## 🧪 テストケース

### 計算例
